# 🤖 Gemini AI Chatbot

A simple AI-powered chatbot built using Google Gemini API and Streamlit. Users can ask questions in natural language and receive intelligent responses generated by Gemini AI in real time.

## 🚀 Features

- Interactive chatbot interface
- Powered by Google Gemini AI
- Real-time response generation
- Simple and clean Streamlit UI
- Environment variable support for API security

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/gemini-ai-chatbot.git
cd gemini-ai-chatbot
```

### 2. Install Dependencies

```bash
pip install streamlit google-generativeai python-dotenv
```

### 3. Configure API Key

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run the Application

```bash
streamlit run main.py
```

## 💡 Usage

1. Open the application in your browser.
2. Enter a question in the text field.
3. Click **Generate Response**.
4. View the AI-generated answer instantly.
